**pixel closet**

An AI-powered personal fashion stylist application built with Python and Streamlit. Users can upload images of individual clothing items to receive personalized aesthetic outfit recommendations generated by Google's Gemini API.

✨ **Features**
* **Visual Analysis:** Uses computer vision through the Gemini API to identify clothing details, colors, and design styles.
* **Subculture Styling:** Specially tailored to generate outfit layouts featuring coquette, Y2K, J-fashion, and casual streetwear aesthetics.
* **Complete Outfit Layouts:** Provides recommendations for matching coordinates (tops/bottoms), footwear, bags, and jewelry accents based on color theory.
* **Custom UI:** Features a sleek holographic UI theme using localized typography overlays (`Soria font.ttf`).

🛠️ **Tech Stack**
* **Language:** Python
* **Frontend/Framework:** Streamlit
* **AI Engine:** Google Gemini API (`gemini-2.5-flash` via the `google-genai` SDK)
